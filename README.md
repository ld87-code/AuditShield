# AuditShield v1: California Lab Compliance Engine

AuditShield is a local-first utility designed for small biotech startups and academic wet labs (teams of ~10) to maintain California-specific safety compliance.

## 🧪 Purpose
Bio-electronics and wet labs in San Francisco face strict oversight from Cal/OSHA and the DTSC. AuditShield automates the generation of mandatory hazardous waste labels and provides a "Safety Timer" to prevent expensive fines.

## 🛠 Key Features
- **CA-Compliant Labeling:** Generates labels meeting **HSC §25200.3.1** standards, including generator info, composition, and physical state.
- **Deadline Tracker:** Calculates mandatory pickup dates based on California's 1-year Satellite Accumulation Area (SAA) rules.
- **Hazard Categorization:** Updated for November 2026 OSHA Hazard Communication standards.
- **Zero-API Infrastructure:** Runs entirely in the browser with no external data transmission, protecting proprietary chemical formulations.

## 🚀 Monetization Strategy
- **Audit-Ready Tooling:** Targeted at seed-stage startups without dedicated Environmental Health and Safety (EH&S) staff.
- **QR Integration Ready:** Designed to be extended with QR codes for instant Safety Data Sheet (SDS) access.

## 📋 Usage
1. Navigate to the `/compliance` directory.
2. Enter waste stream details and the accumulation start date.
3. Use the "Days Remaining" dashboard to monitor compliance.
4. Print directly to a standard label printer.
